INSTALLATION INSTRUCTIONS:

1. Download and install node.js from this website: https://nodejs.org/en/download/

CONFIGURATION INSTRUCTIONS:

1. Edit input_dir or output_path variables in "gen_csv.cmd"
   - input_dir is the path to the folder full of JSON files
   - output_path is the path to the CSV file you wish to output

HOW TO RUN

1. Run gen_csv.cmd
